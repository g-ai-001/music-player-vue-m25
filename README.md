# music-player-vue-m25

基于 Vue3 的音乐播放器网页应用

## 功能特性

- 左侧音乐播放列表
- 右侧播放器界面
  - 歌曲图片展示
  - 歌曲信息显示
  - 播放控制按钮
  - 滚动歌词显示
  - 歌曲进度条
- 歌曲图片暗色高斯模糊背景
- 响应式布局（支持手机、平板、PC）

## 技术栈

- Vue 3 (Composition API)
- TypeScript
- Vite
- CSS3
